🛠️ API Endpoints Available

The CSV validation system provides these endpoints:

📋 Format Documentation:
GET /csv/csv-formats # Get all CSV format specs GET /csv/csv-formats/{type} # Get specific format (inventory/dishes/sales)
📥 Template Downloads:
GET /csv/csv-template/{type} # Download CSV template with sample data
🔍 Validation:
POST /csv/validate-csv/{type} # Validate with authentication POST /csv/test-validate-csv/{type} # Test validation (no auth required)
